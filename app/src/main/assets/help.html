FakeLock is an app for extra privacy. It hides selected apps and keeps them hidden until master password is entered. It doesn't disable default Keuguard rather acts as secondary lock screen.
How to Use:

Select all apps that you want to hide from Home tab.

Set primary and secondary passwords from Settings tab.

Whenever LockScreen Activity pop-ups, double tap on quote to enable password input.
